## 💡 What Changed

### Code Before:
```javascript
const handleGetStarted = () => {
  if (user) navigate("/dashboard");
  else navigate("/auth");
};
// If error → "something went wrong" (generic)
```

### Code After:
```javascript
const handleGetStarted = () => {
  try {
    if (user) navigate("/dashboard", { replace: false });
    else navigate("/auth", { replace: false });
  } catch (error) {
    console.error("Navigation error:", error);
    toast.error("Failed to navigate. Please try again.");
  }
};
// If error → "Failed to navigate. Please try again." (specific)
```
