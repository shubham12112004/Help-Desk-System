💡 TROUBLESHOOTING
================================================================================

If login fails:
1. Check internet connection
2. Verify Supabase connection status (green indicator on page)
3. Ensure email format is correct
4. Try clearing browser cache (Ctrl+Shift+Delete)
5. Check if account exists (try signup instead)

If password field not showing eye icon:
1. Hard refresh browser (Ctrl+Shift+R)
2. Check browser console for errors (F12)
3. Ensure JavaScript is enabled
4. Try different browser

If getting "Email rate limited" error:
1. Wait 1 hour before trying again
2. Use Magic Link option instead
3. Disable email confirmation in Supabase settings
