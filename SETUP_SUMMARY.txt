```
Go to: http://localhost:5174

Click: "Don't have an account? Sign up"

Fill in:
  Email:     admin@hospital.local
  Password:  Admin@123456
  Full Name: Admin Dashboard
  Role:      Staff

Click: Sign Up
```

### STEP 2: Make Admin in Database (2 minutes)
```
Go to: https://app.supabase.com → Your Project → SQL Editor

Run:
  SELECT id FROM auth.users WHERE email = 'admin@hospital.local';
  
  UPDATE public.user_roles 
  SET role = 'admin' 
  WHERE user_id = 'PASTE_ID_HERE';
```

### STEP 3: Login to Admin Dashboard (2 minutes)
```
1. Press F12 > Application > Clear All Storage
2. Refresh: http://localhost:5174
3. Sign In with:
   Email:    admin@hospital.local
   Password: Admin@123456
4. Click "Admin" in sidebar
5. See Admin Dashboard ✓
```
